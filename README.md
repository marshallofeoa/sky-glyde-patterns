# Sky-Glyde Design Patterns Database

A comprehensive interactive database of design patterns for the Sky-Glyde aerial mobility platform. Explore patterns across categories, platforms, and priority levels with detailed implementation guidance.

## Features

- **Interactive filtering** by category, platform, and priority
- **Comprehensive pattern library** covering onboarding, safety, trust, pricing, and more
- **Implementation details** specific to Sky-Glyde
- **Adoption metrics** for each pattern
- **Real-time search and filtering**

## Categories

- Safety Messaging
- Trust Building
- Onboarding
- Terminology
- Pricing

## Platforms

- Ground (autonomous vehicles)
- Aerial (eVTOL)
- Both

## Getting Started

### Local Development

Simply open `public/index.html` in your browser, or serve with:

```bash
npx http-server public
```

### Deployment

This project is deployed on Vercel and updates automatically on push to main.
